安裝過程很簡單，最簡單的自動化安裝只需要點一下按鈕，執行一個命令即可
對於計算機專家也可以按照說明選擇手動安裝
以下步驟假定：
用戶主目錄為(/home/yourname)，瀏覽器默認下載目錄為(/home/yourname/Downloads)
請根據實際情況進行相應修改

自動化安裝步驟

  1. 點擊初始化按鈕，記住打開的下載目錄路徑
  2. windows使用者在打開的下載目錄裡找到scrapbookq目錄，執行scrapbookq目錄裡面的init_scrapbookq.bat批次檔
  3. linux使用者在控制台執行以下命令(替換 yourname/Downloads為打開的下載目錄路徑)：

    cd /home/ yourname/Downloads/scrapbookq;chmod +x scrapbookqmsg;./scrapbookqmsg init

  4. 重新啟動firefox瀏覽器

簡單安裝步驟

  1. 點擊初始化按鈕
  2. 移動 /home/yourname/Downloads/scrapbookq/native-messaging-hosts 到 ~/.mozilla/
  3. 編輯 native-messaging-hosts/scrapbookqmsg.json，修改"path"
  4. 編輯 ~/Downlaods/scrapbookq/scrapbookq.conf，修改rdfpath
  5. chmod +x scrapbookqmsg;./scrapbookqmsg

詳細安裝步驟

  1. 安裝ScrapbookQ插件，能看到此文檔說明已經初步安裝好了
  2. 點擊按鈕初始化ScrapbookQ文件夾和文件(/home/yourname/Downloads/scrapbookq)
  3. 打開scrapbookeq目錄，移動(native-messaging-hosts)文件夾到firefox配置文件夾裏(/home/yourname/.mozilla/)
  4. 編輯(native-messaging-hosts)文件夾裏的文件(scrapbookqmsg.json)
  5. 修改(scrapbookqmsg.json)裏面"path"一項的值為ScrapBook文件夾裏scrapbookqmsg文件的路徑
  6. 修改完成後大概這個樣子：("path": "/home/yourname/Downloads/scrapbookq/scrapbookqmsg")
  7. 編輯scrapbookq目錄裏的配置文件scrapbookq.conf，將rdfpath的值修改為ScrapBook目錄的路徑
  8. 修改完成後大概這個樣子：(rdfpath = /home/windowsD/Documents/firefox/ScrapBook/)
  9. 最後設置本地程序文件可執行屬性：chmode +
  10. 修改保存文件後重新啟動瀏覽器會看到側邊欄裏的ScrapbookQ目錄

安裝配置完成後文件夾及文件

  • 文件夾及文件
    1. /home/yourname/Downloads/scrapbookq/
    2. /home/yourname/Downloads/scrapbookq/scrapbookq.conf
    3. /home/yourname/Downloads/scrapbookq/scrapbookq.rdf
    4. /home/yourname/Downloads/scrapbook
    5. /home/yourname/.mozilla/native-messaging-hosts/
    6. /home/yourname/.mozilla/native-messaging-hosts/scrapbookqmsg.json
  • 配置文件
    • /home/yourname/Downloads/scrapbookq/scrapbookq.conf
    • /home/yourname/.mozilla/native-messaging-hosts/scrapbookqmsg.json

ScrapbookQ 使用說明

  1. 安裝ScrapbookQ插件後，會在瀏覽器工具欄添加一個ScrapbookQ圖標
  2. 鼠標左鍵點擊工具欄ScrapbookQ圖標 會在瀏覽器側邊欄打開ScrapbookQ目錄
  3. 如果瀏覽器側邊欄裏沒有出現ScrapbookQ目錄請在瀏覽器側邊欄頂部下拉列表裏選中ScrapbookQ
  4. 全新安裝ScrapbookQ插件後需要初始化ScrapbookQ文件夾和文件：
  5. 點擊按鈕完成初始化後，會在用戶瀏覽器默認下載目錄裏新建scrapbook文件夾及必要的配置文件和本地程序文件
  6. 在瀏覽器側邊欄裏打開的ScrapbookQ目錄頁面上，可以通過鼠標右鍵菜單管理ScrapbookQ目錄
  7. 安裝ScrapbookQ插件後，在瀏覽器鼠標右鍵菜單會出現ScrapbookQ菜單和菜單項：
    • 保存頁面： 保存當前打開的頁面到ScrapbookQ目錄裏
    • 切換側邊欄： 打開/關閉瀏覽器側邊欄
    • 打開下載目錄： 打開瀏覽器默認下載目錄，ScrapbookQ目錄就放在這裏
  8. ScrapbookQ目錄頁面上可以進行以下操作：
    鼠標左鍵 在當前標簽頁打開點擊對象的ScrapbookQ文件鏈接
    右鍵菜單 在新建標簽頁中打開 在新建標簽頁打開點擊對象的ScrapbookQ文件鏈接
    打開源鏈接 在當前標簽頁打開點擊對象的初始URL鏈接
    新建文件夾 在點擊對象之前新建文件夾
    屬性 查看點擊對象的屬性
    剪切 剪切點擊對象的文件鏈接或者文件夾
    粘貼 粘貼已剪切的文件鏈接或者文件夾到點擊對象之前/之內
    刪除 刪除點擊對象的文件鏈接或者文件夾
  9. 隨時可以點擊瀏覽器工具欄的ScrapbookQ圖標 激活本頁面，也可以右擊圖標激活右鍵菜單

導入ScrapBook數據

  1. 修改scrapbookq/scrapbookq.conf
  2. 在rdfpath值的末尾添加ScrapBook數據文件scrapbook.rdf所在路徑
  3. 大概這個樣子：rdfpath = /home/windowsD/Documents/firefox/ScrapBook/
  4. 重新啟動firefox瀏覽器、如果原有ScrapBook數據比較多，第一次讀取數據會有點慢
