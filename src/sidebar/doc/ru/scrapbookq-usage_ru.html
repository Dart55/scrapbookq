Руководство по установке ScrapbookQ

Добро пожаловать в ScrapbookQ
Процесс установки очень прост, простейшей автоматической установке нужно только нажать кнопку, выполнить команду
Для компьютерных экспертов следуйте инструкциям, которые необходимо выбрать для установки вручную.
Следующие шаги предполагают:
Домашний каталог пользователя (/home/yourname)，каталог загрузки браузера по умолчанию(/home/yourname/Downloads)
Просьба внести соответствующие изменения в соответствии с фактической ситуацией

Автоматизация шагов установки

  1. Нажмите кнопку инициализации, не забудьте открыть путь к каталогу загрузки
  2. Выполните следующую команду на консоли (замените yourname/Downloadsоткрытым каталогом каталога загрузки):

    cd /home/ yourname/Downloads/scrapbookq;chmod +x initscrapbookq.py;python3 initscrapbookq.py

  3. Перезапустить браузер Firefox

Простая процедура установки

  1. Нажмите кнопку «Инициализировать»
  2. Переместить /home/yourname/Downloads/scrapbookq/native-messaging-hosts в ~/.mozilla/
  3. редактировать native-messaging-hosts/scrapbookqmsg.json，модификация"path"
  4. редактировать ~/Downlaods/scrapbookq/scrapbookq.conf，модификацияrdfspath
  5. chmod +x init.sh; sh init.sh

Инструкции ScrapbookQ для использования

  1. После установки плагина ScrapbookQ значок значка ScrapbookQ добавляется на панель инструментов браузера
  2. Левый щелчок мышью на панели инструментов Icon ScrapbookQ Каталог ScrapbookQ открывается на боковой панели браузера
  3. Если ScrapbookQ не отображается на боковой панели браузера, выберите ScrapbookQ из раскрывающегося списка в верхней части боковой панели браузера
  4. После новой установки плагина ScrapbookQ вам необходимо инициализировать папки и файлы ScrapbookQ:
  5. Нажмите кнопку, чтобы завершить инициализацию, каталог загрузки по умолчанию в браузере пользователя, создать новую папку с записками и необходимые файлы конфигурации и файлы локальных программ
  6. Страница каталога ScrapbookQ, открытая на боковой панели браузера, директория ScrapbookQ может управляться с помощью меню правой кнопки мыши
  7. После установки плагина ScrapbookQ меню ScrapbookQ и пункты меню отображаются в меню правой кнопки мыши браузера:
    • Сохранить страницу: сохранить открытую страницу в настоящее время в каталоге ScrapbookQ
    • Переключить боковую панель: открывает / закрывает боковую панель браузера
    • Откройте каталог загрузки: откройте каталог загрузки по умолчанию в браузере, каталог ScrapbookQ находится здесь
  8. Страница каталога ScrapbookQ позволяет вам сделать следующее:
    Левая кнопка мыши Откройте ссылку на файл ScrapbookQ объекта click на текущей вкладке
    Меню правой кнопки мыши Открыть в новой вкладке Откройте ссылку на файл ScrapbookQ объекта click на вкладке «Создать»
    Открыть ссылку источника Откройте начальную ссылку URL-адреса щелкнутого объекта на текущей вкладке
    Создать новую папку Создайте новую папку, прежде чем нажимать на объект
    собственности Просмотр свойств объекта click
    ножницы Вырезать ссылку или папку файла объекта
    ручка Вставить ссылку или папку файла разреза до / внутри объекта клика
    удалять Удалить ссылку или папку файла щелкнутого объекта
  9. В любой момент вы можете щелкнуть значок панели инструментов ScrapbookQ, чтобы активировать эту страницу, вы также можете щелкнуть правой кнопкой мыши значок, чтобы активировать контекстное меню
