Automatic Installation

  1. Click the "initialization" button to automatically open the download directory where scrapbookq is located
  2. The windows user finds the "scrapbookq" directory in the open download directory and executes the "init_scrapbookq.bat" batch file in the "scrapbookq" directory
  3. Linux user execute the following command at console(Replace yourname/Downloadswith the open directory path)：

    cd /home/ yourname/Downloads/scrapbookq;chmod +x scrapbookqmsg;./scrapbookqmsg init

  4. Restart firefox

ScrapbookQ Manual

  1. After the installation, the ScrapbookQ icon button is added to the Firefox toolbar.
  2. Click the toolbar ScrapbookQ icon button will open the ScrapbookQ Manual page in the Firefox browser
  3. If there is no ScrapbookQ directory in the browser side sidebar, please select ScrapbookQ in the top drop-down list of the browser side sidebar.
  4. After the new installation of the ScrapbookQ plug-in, you need to initialize the ScrapbookQ folder and files throuth
  5. After click the button, the scrapbook folder, the necessary configuration files and the native application will be created in the default download directory of the Firefox browser.
  6. Through right click menu of ScrapbookQ in sidebar, you can manage saved web pages.
  7. ScrapbookQ right click menu items:
    • CapturePage: capture current web page and save it to ScrapbookQ
    • ToggleSidebar: open/close browser sidebar
    • OpenDownloadDir: Open the browser's default download directory, in which the scrapbookq directory stored.
  8. Operation in ScrapbookQ：
    Left Click Open clicked ScrapbookQ page in current Tab
    Right Click Menu Open In New Tab Open clicked ScrapbookQ page in a new Tab
    Open origin URL Open clicked ScrapbookQ page's source link in current Tab
    New Directory Create a new directory
    Property Show clicked ScrapbookQ page's property
    Cut Cut clicked ScrapbookQ page or directory
    Paste Paste the cut file link or folder to the front of the click target. If the target is a directory, it is inserted into the directory.
    Delete Delete the click target, if the target is a directory, and delete all the files in the directory.
  9. You can click the ScrapbookQ toolbar icon button to open this manual page.

Importing ScrapBook data

  1. Modify scrapbookq\scrapbookq.conf
  2. Add the full path of the RDF to the rdfpath
  3. Looks like: rdfpath=D:\\Document\\ScrapBook\\
  4. If the scrapbook.rdf file is large, it will take a long time to import the ScrapBook data for the first time. Please be patient.
