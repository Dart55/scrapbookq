Automatic Installation

  1. Click the button，remember the opened directory path
  2. Windows user finds scrapbookq directory in open download directory, executes init_scrapbookq.bat batch file in scrapbookq directory
  3. Linux user execute the following command at console(Replace yourname/Downloadswith the opened directory path)：

    cd /home/ yourname/Downloads/scrapbookq;chmod +x scrapbookqmsg;./scrapbookqmsg init

  4. Restart firefox

ScrapbookQ Manual

  1. After Installlation, you can find the ScrapbookQ icon at firefox toolbar.
  2. Left click on ScrapbookQ toolbar icon will open sidebar which containing ScrapbookQ
  3. Click on the top of sidebar and select ScrapbookQ item when ScrapbookQ disappeared.
  4. First time install ScrapbookQ will init scrapbookq folder:
  5. Initialization will crate a folder named scrapbookq in browser default download dir, and create some files.
  6. Through right click menu on ScrapbookQ in sidebar, you can manage saved web pages.
  7. ScrapbookQ right click menu items:
    • CapturePage: capture current web page and save it to ScrapbookQ
    • ToggleSidebar: open/close browser sidebar
    • OpenDownloadDir: open brower defautl download folder which also containing scrapbookq folder
  8. Operation in ScrapbookQ：
    Left Click Open clicked ScrapbookQ page in current Tab
    Right Click Menu Open In New Tab Open clicked ScrapbookQ page in a new Tab
    Open origin URL Open clicked ScrapbookQ page's link in current Tab
    New Directory Create a new directory
    Property Show clicked ScrapbookQ page's property
    Cut Cut clicked ScrapbookQ page or directory
    Paste Insert cut ScrapbookQ page or directory before/into clicked ScrapbookQ page/directory
    Delete Delete clicked ScrapbookQ page or directory and all files in it
  9. You can active this manual through left click on ScrapbookQ toolbar icon. , or active ScrapbookQ menu by write click.

Import ScrapBook Data

  1. Modify scrapbookq\scrapbookq.conf
  2. add real path of ScrapBook\scrapbook.rdf at the end of rdfpath
  3. Looks like: rdfpath=D:\\Document\\ScrapBook\\
  4. Restart firefox. The first time import ScrapBook rdf file may be slow if the rdf file is big
