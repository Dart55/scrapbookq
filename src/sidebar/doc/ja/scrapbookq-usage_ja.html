自動インストールの手順

  1. [初期化] ボタン，ダウンロードディレクトリのパスを開くことを忘れないでください
  2. Windows のユーザーが見つかりました scrapbookq ディレクトリを開いているダウンロードディレクトリ, 実行 init_scrapbookq.bat バッチファイルの scrapbookq ディレクトリ
  3. Linux ユーザー 在コンソールは次のコマンドを実行します。(交換 yourname/Downloadsダウンロードディレクトリのパスを開く)：

    cd /home/ yourname/Downloads/scrapbookq;chmod +x scrapbookqmsg;./scrapbookqmsg init

  4. Firefox ブラウザを再始動しなさい

ScrapbookQ 使用方法

  1. SCRAPBOOKQ プラグインをインストールすると、ブラウザのツールバーに SCRAPBOOKQ アイコンが追加されます。
  2. マウスの左ボタンをクリックしてツールバーの scrapbookq アイコン ブラウザのサイドバーで SCRAPBOOKQ ディレクトリを開きます。
  3. SCRAPBOOKQ ディレクトリがブラウザのサイドバーに表示されない場合は、サイドバーの上部にあるドロップダウン・テーブルで SCRAPBOOKQ を選択します。
  4. SCRAPBOOKQ フォルダとファイルは、SCRAPBOOKQ プラグインの新しいインストール後に初期化する必要があります：
  5. ボタンをクリックして初期化を完了すると、ユーザーブラウザのデフォルトのダウンロードディレクトリには、新しいスクラップブックフォルダと必要な設定ファイルとローカルプログラムファイルを作成することになります
  6. ブラウザのサイドバーに表示される SCRAPBOOKQ ディレクトリページで、マウスの右ボタンメニューから SCRAPBOOKQ ディレクトリを管理できます。
  7. SCRAPBOOKQ プラグインをインストールすると、SCRAPBOOKQ メニューとメニュー項目がブラウザのマウスの右クリックメニューに表示されます。：
    • ページを保存: SCRAPBOOKQ ディレクトリに現在開いているページを保存します。
    • サイドバーの切り替え: ブラウザのサイドバーを開く/閉じる
    • オープンダウンロードディレクトリ: オープンブラウザのデフォルトのダウンロードディレクトリ、SCRAPBOOKQ ディレクトリはここに配置されます
  8. ScrapbookQ[カタログ] ページでは、次の操作を実行できます。：
    マウスの左ボタン 現在のタブで Scrapbookq ファイルリンクを開く
    右クリックメニュー 新しいタブで開く [新規] タブで、クリックオブジェクトの Scrapbookq ファイルリンクを開きます。
    オープンソースリンク 現在のタブページでオブジェクトの初期 URL リンクを開く
    新しいフォルダ オブジェクトをクリックする前に新しいフォルダを作成する
    プロパティ クリックしたオブジェクトのプロパティを表示する
    せん断 オブジェクトのファイルリンクまたはフォルダを切り取る
    貼り 切り取りファイルのリンクまたはフォルダを、クリックオブジェクトの前/内側に貼り付ける
    削除 クリックしたオブジェクトのファイルリンクまたはフォルダを削除する
  9. あなたは、ブラウザ・ツールバーで SCRAPBOOKQ アイコンをいつでもクリックすることができます このページをアクティブにするか、アイコンを右クリックして右クリックメニューをアクティブにします。

スクラップブックデータのインポート

  1. 変更scrapbookq\scrapbookq.conf
  2. Rdfpath 値スクラップブックの末尾にスクラップブックデータファイルを追加します。 RDF パス
  3. このように。(windows): rdfpath=C:\\111\\ScrapBook\\;D:\\222\\ScrapBook\\;E:\\333\\ScrapBook\\
  4. このように。( linux ): rdfpath=/111/ScrapBook/;/222/ScrapBook/;/333/ScrapBook/;
  5. 元のスクラップブックのデータがより多くの場合、Firefox のブラウザを再起動すると、データを読み取るために最初の時間は少し遅いです
